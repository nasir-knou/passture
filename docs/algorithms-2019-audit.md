# Algorithms 2019 Past Exam Audit

알고리즘 2019 기말(`data/subjects/algorithms/past-exams-2019.yaml`)을 원본 이미지(`origin/19-알고리즘-3학년-3교시-1.png`, `origin/19-알고리즘-3학년-3교시-2.png`) 및 정답표(`origin/정답.txt`)와 10문항 단위로 대조한 기록이다.

## 기준

- 원본 1~35번의 prompt, 선택지, 공통 지문이 YAML의 `e19-01`~`e19-35`에 같은 의미로 들어가 있어야 한다.
- 공식 정답표의 알고리즘 2019 답안열 `32134421212434432234123324421444312`와 YAML `answers`가 일치해야 한다.
- 해설은 정답 근거, 각 오답의 이유, 오답 시 필요한 개념 설명, 문제 핵심 개념 요약을 포함해야 한다.
- 표, 그래프, 코드처럼 구조화 가능한 자료는 passage/diagram/code로 작성하고, 불가피한 경우에만 이미지를 사용한다.
- 검증 단위는 원본 문항 번호 기준 10문항이다.

- 문항 수: 35문항.
- YAML ID 범위: `e19-01`~`e19-35`, 원본 1~35번에 순서 대응.
- 정답 대조: YAML 답안열 `32134421212434432234123324421444312`, 공식 정답표와 일치.
- passage 사용: text/code, `data-table`, `simple-graph` diagram.
