# Algorithm Textbook Audit

알고리즘 기본서 문제(`data/subjects/algorithms/textbook.yaml`)를 원본 이미지(`origin/알고리즘 교재`)와 10문항 단위로 대조한 기록이다.

## 기준

- 원본 문제의 prompt, 선택지, 정답이 YAML에 같은 의미로 들어가 있어야 한다.
- 주관식/서술형 문항은 제외한다. 알고리즘 기본서 원본 대조 범위에는 주관식 문항이 없었다.
- 원본 해설이 짧은 경우에도 YAML 해설은 정답 근거, 각 오답의 이유, 핵심 개념 요약을 포함해야 한다.
- 수식, 배열, 그래프 선택지, 알고리즘 의사코드는 가능한 한 텍스트/수식/간선 집합으로 구조화하고, 원본 시각 형태 자체가 채점 단서인 경우에만 이미지 참조를 사용한다.
- 검증 단위는 YAML 배열 순서 기준 10문항이다.
