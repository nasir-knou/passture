# Artificial Intelligence 2018 Past Exam Audit

인공지능 2018 기말(`data/subjects/artificial-intelligence/past-exams-2018.yaml`)을 원본 이미지(`origin/18-인공지능-4학년-1교시-1.png`, `origin/18-인공지능-4학년-1교시-2.png`) 및 정답표(`origin/정답.txt`)와 10문항 단위로 대조한 기록이다.

## 기준

- 원본 1~35번의 prompt, 선택지, 공통 지문이 YAML의 `e18-01`~`e18-35`에 같은 의미로 들어가 있어야 한다.
- 공식 정답표의 인공지능 2018 답안열 `13413242412431342134124213342131224`와 YAML `answers`가 일치해야 한다.
- 해설은 정답 근거, 각 오답의 이유, 오답 시 필요한 개념 설명, 문제 핵심 개념 요약을 포함해야 한다.
- 코드, 표, 그래프처럼 구조화 가능한 자료는 passage/diagram으로 작성하고, 현재 diagram으로 재현하기 어려운 경우에만 이미지를 사용한다.
- 검증 단위는 원본 문항 번호 기준 10문항이다.

## 검증 결과

- 문항 수: 35문항.
- YAML ID 범위: `e18-01`~`e18-35`, 원본 1~35번에 순서 대응.
- 정답 대조: YAML 답안열 `13413242412431342134124213342131224`, 공식 정답표와 일치.
- passage 사용:
  - `g18-minimax-tree-01`: `simple-graph` diagram.
- 이미지 사용:
  - `e18-29` 선택지 1~4: 원본 산점도 선택지를 `public/images/subjects/artificial-intelligence/past-exams/2018/e18-29-choice-*.png`로 분리 보존.
- 해설 구조: 35문항 모두 `선택지 1:`~`선택지 4:` 근거와 `핵심 개념:` 섹션 포함.
- 데이터 검증: 사용자 정의 YAML 검사로 문항 수, 정답열, 해설 구조, 이미지 경로 존재 여부 통과.

## 발견 사항

- 기존 2018 YAML의 정답열은 공식 정답표와 일치했으나, 전체 해설이 선택지별 근거와 핵심 개념 섹션을 갖추지 못해 35문항 모두 해설을 보강했다.
- 29번은 원본 선택지가 산점도 이미지이므로, 텍스트 설명만으로는 원본 동일성 검증 기준이 약했다. 해당 선택지는 원본에서 잘라낸 이미지와 대체 텍스트를 함께 연결했다.
- 현재 전체 `pnpm data:build`는 인공지능 2018 데이터가 아니라 기존 `data/subjects/algorithms/workbook.yaml`의 빈 data-table 열 문제로 중단된다.
