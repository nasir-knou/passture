# Artificial Intelligence 2019 Past Exam Audit

인공지능 2019 기말(`data/subjects/artificial-intelligence/past-exams-2019.yaml`)을 원본 이미지(`origin/19-인공지능-4학년-1교시-1.png`, `origin/19-인공지능-4학년-1교시-2.png`) 및 정답표(`origin/정답.txt`)와 10문항 단위로 대조한 기록이다.

## 기준

- 원본 1~35번의 prompt, 선택지, 공통 지문이 YAML의 `e19-01`~`e19-35`에 같은 의미로 들어가 있어야 한다.
- 공식 정답표의 인공지능 2019 답안열 `41243113423231213442124134123343214`와 YAML `answers`가 일치해야 한다.
- 해설은 정답 근거, 각 오답의 이유, 오답 시 필요한 개념 설명, 문제 핵심 개념 요약을 포함해야 한다.
- 코드, 표, 그래프처럼 구조화 가능한 자료는 passage/diagram으로 작성하고, 현재 diagram으로 재현하기 어려운 경우에만 이미지를 사용한다.
- 검증 단위는 원본 문항 번호 기준 10문항이다.
