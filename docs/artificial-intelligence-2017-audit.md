# Artificial Intelligence 2017 Past Exam Audit

인공지능 2017 기말(`data/subjects/artificial-intelligence/past-exams-2017.yaml`)을 원본 이미지(`origin/17-인공지능-4학년-1교시-1.png`, `origin/17-인공지능-4학년-1교시-2.png`) 및 정답표(`origin/정답.txt`)와 10문항 단위로 대조한 기록이다.

## 기준

- 원본 1~35번의 prompt, 선택지, 공통 지문이 YAML의 `e17-01`~`e17-35`에 같은 의미로 들어가 있어야 한다.
- 공식 정답표의 인공지능 2017 답안열 `14223412133241241412334123324214133`과 YAML `answers`가 일치해야 한다.
- 해설은 정답 근거, 각 오답의 이유, 오답 시 필요한 개념 설명, 문제 핵심 개념 요약을 포함해야 한다.
- 코드, 표, 그래프처럼 구조화 가능한 자료는 passage/diagram으로 작성하고, 현재 diagram으로 재현하기 어려운 경우에만 이미지를 사용한다.
- 검증 단위는 원본 문항 번호 기준 10문항이다.

## 검증 결과

- 문항 수: 35문항.
- YAML ID 범위: `e17-01`~`e17-35`, 원본 1~35번에 순서 대응.
- 정답 대조: YAML 답안열 `14223412133241241412334123324214133`, 공식 정답표와 일치.
- passage 사용:
  - `g17-cnf-clauses-08`: text.
  - `g17-production-rules-15`: text.
  - `g17-fuzzy-sets-17`: text.
  - `g17-connected-grid-19`: text.
  - `g17-feature-purpose-24`: text.
- 해설 구조: 35문항 모두 `선택지 1:`~`선택지 4:` 근거와 `핵심 개념:` 섹션 포함.
- 데이터 검증: 사용자 정의 YAML 검사와 `validateQuestionFile` 개별 스키마 검사 통과.

## 발견 사항

- 인공지능 2017 기말 YAML이 존재하지 않아 새로 추가했다.
- 2017 기말을 앱에서 선택할 수 있도록 `data/catalog.yaml`에 `past-exams-2017` 항목을 추가했다.
- 현재 전체 `pnpm data:build`는 인공지능 2017 데이터가 아니라 기존 `data/subjects/algorithms/workbook.yaml`의 빈 data-table 열 문제로 중단된다.
