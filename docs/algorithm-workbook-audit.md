# Algorithm Workbook Audit

알고리즘 워크북(`data/subjects/algorithms/workbook.yaml`)을 원본 이미지(`origin/알고리즘 워크북`)와 10문항 단위로 대조한 기록이다.

## 기준

- 원본 문제의 prompt, 선택지, 정답이 YAML에 같은 의미로 들어가 있어야 한다.
- 원본 해설이 짧은 경우에도 YAML 해설은 정답 근거, 각 오답의 이유, 핵심 개념 요약을 포함해야 한다.
- 도표화 가능한 내용은 `diagram`/텍스트 구조로 유지하고, 원본 이미지가 채점 단서인 경우에만 이미지 참조를 사용한다.
- 검증 단위는 YAML 배열 순서 기준 10문항이다.
