# Discrete Math Lecture Exercises Audit

대상: `data/subjects/discrete-math/lecture-exercises.yaml`

원본: `origin/이산수학 강의연습문제`

검토 기준:

- `docs/data-schema.md`의 강의 문제 ID, 지문/이미지 처리 규칙
- 기존 기출 데이터셋의 문제, 선택지, 정답, 해설 작성 방식
- 원본 이미지를 10문항 단위로 대조

## 결과

- 총 69문항이 원본 1강~15강 69개 PNG와 대응한다.
- 문제 본문, 선택지, 정답은 원본과 대조해 YAML에 직접 입력했다.
- 반복되는 코드, 행렬, 진리표, 그래프, 오토마타, 빈칸 지문은 가능한 `passages`와 `diagram`으로 구조화했다.
- 선택지별 해설과 `핵심 개념:`을 69문항 전체에 보강했다.
- 잔여 이미지는 `l09-03` 1건이다. 네 개의 동형 그래프 후보를 한 화면에서 시각 비교해야 하는 문항이라 원본 그림의 위상과 배치를 보존했다.

## 1~10번

범위:

- `l01-03` ~ `l03-01`
- 원본 `1강/003.png` ~ `3강/001.png`

확인 내용:

- 의사코드, 추상화, 알고리즘 표현, 명제, 논리연산, 진리표, 조건명제, 동치명제를 대조했다.
- 의사코드는 `g01-03-pseudocode` 코드 지문으로 분리했다.
- 진리표 선택지는 이미지 대신 `data-table` diagram으로 입력했다.
