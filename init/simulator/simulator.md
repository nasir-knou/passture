# 모의 시험 페이지 개발 계획

실제 시험환경 UI에서, 등록된 문제 출처를 시간에 맞춰 풀어보는 서비스임

참고) 공식 튜토리얼 https://ibt.knou.ac.kr/js/site/examne/tutorial/tutorial/start.html

크게 아래의 3가지 항목으로 구성 
- 대상과목/출제범위 선정
- 시험 화면
- 결과 화면

## 대상과목/출제범위 선정
- 등록된 과목 중 시험형식으로 테스트 할 3개의 과목을 선택
  - 각각 과목에서 시험에 사용할 출처 선택
  - 전체 문제를 풀지 무작위 추출 문제를 풀지 선택 옵션 필요
    - 무작위 추출 옵션 선택시 b02, l03 같은 문제id의 몇강인지 참조해 균일하게 분포되도록 25개의 문제 선정
- 과목당 25개의 문제, 25분의 시험 시간
  - 2개의 과목을 선택했으면 제한시간 50분 등 자동계산하여 반영

## 시험 화면
- simulator 디렉토리에 동봉된 png 확인
- 제일 상단바에는 방통대로고와 함께 20XX년 X학기 기말 시험 과 시험실 퇴실버튼, 우측에는 남은시간 표시
- 그아래 시험화면에는 문제풀이 영역과 정답기록 영역이 있음
  - 문제풀이 영역에는 교과목명(푼 문제 수/전체 문제 수) 표시 (최대3과목)
  - 정답기록 영역에는 시험시간, 시험시작/종료시간, 응시자 정보
    - 시험시간은 과목수에 따라 25분/50분/75분
    - 시작시간은 13:30분으로 고정, 종료시간은 시험시간에 맞춰 생성
    - 응시자 정보는 [모의테스트] 20nnnn-123456 방송대 
- 정답기록 영역의 문제번호를 누르면 문제풀이 영역의 문제로 바로 이동
- 문제풀이 영역의 답안을 누르면 정답기록 영역에 자동으로 저장됨
- 다시확인할 문제는 책갈피 버튼을 눌러 표시할 수 있으며 정답기록 영역에도 책갈피표시를 볼 수 있음
- 상단의 교과목명을 눌러 과목을 이동할 수 있음, 이때 정답 입력이나 책갈피 기록은 유지
- 시간이 종료되거나 시험실 퇴실 버튼을 누르고 확인을 누르면 종료

## 결과 화면
- 기존 문제풀이의 결과화면과 유사하지만 상단에  교과목명 정도 눌러서 이동할수있게하면 되지않을까
