# 변환 제외 문항

아래 문항은 현재 PASSture 문제 스키마로 안정적으로 변환하기 어려워 YAML 세트에서 제외한다.

제외 기준:

- 주관식/서술형 문항
- 단일정답 4지선다 또는 OX가 아닌 문항
- 선택지가 5개 이상이거나 보기 조합형처럼 앱에서 독립 선택지로 표현하기 어려운 문항

- 원본 캡처에서 정답 또는 해설 확인이 어려운 문항
